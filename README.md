## Getting Started

### Usage

Clone the app:
``````
git clone https://github.com/byrdsandbytes/capContactsDemo.git
cd capContactsDemo
``````

Install dependencies:
``````
npm install
``````

build the App:
```
npx ionic build
```

Sync build to Android and ios Platforms
`````
npx cap sync
`````

open Xcode or Android Studio project

```
npx cap open ios 
npx cap open android
```
